# CardCraft 🧵✨

CardCraft is an AI-ready MVP web app for generating marketplace product card content for clothing items.

## Features 🚀

- Product card content generation (title, description, bullets, CTA)
- Image prompts in Russian and English
- Style/tone presets
- Local project saving in `localStorage`
- Optional product photo upload
- Export to JSON and TXT
- Copy and regenerate per section

## Tech Stack 🛠️

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Quick Start ⚡

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
